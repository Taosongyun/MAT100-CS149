01Overview

This repository collects readable course-style pages. Each page is designed as a self-contained handout: definitions first, then examples, then worked solutions, then practice problems.

Design Principle The goal is not just to store notes. The goal is to make each topic navigable, visually structured, and easy to review before homework, exams, or recitations.
Current Status Week 1 is ready. Additional weeks can be added by placing new HTML files in the repository root and linking them from this index page.

04How to Use This Repository

  • Open a weekly note from the card grid above.
  • Read the definitions before jumping into examples.
  • Use the worked examples to check the logic of each proof step.
  • Try the practice questions before revealing solutions.
