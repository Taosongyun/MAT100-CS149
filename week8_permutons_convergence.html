01Why We Care

Last week introduced permutons as limit objects for permutation sequences. This week makes them concrete: we learn to compute pattern densities, see exactly how a permutation turns into a permuton via its matrix, define convergence rigorously via CDFs, and work through two important examples — the identity sequence (converges to the diagonal) and the random walk (has a specific non-uniform permuton limit). We also prove the basis theorem for permutation classes, tying together the pattern avoidance story from Week 7.

02Pattern Densities

Definition — Pattern Density For a permutation σ ∈ Sₙ and a pattern π ∈ Sₖ with k ≤ n:
p(π, σ) = (# k-element subsets of positions realizing π in σ) / C(n, k)
This is the probability that a uniformly random k-element subset of {1,…,n} forms pattern π in σ.
Worked Example — density(132, 6214537)

σ = (6,2,1,4,5,3,7), n=7, k=3. Total triples: C(7,3) = 35.

Count k-element subsets where the relative order is 132 (medium, large, small):

  • Positions (1,2,4): values (6,2,4) → ranks (3,1,2) = 312. ✗
  • Positions (2,4,6): values (2,4,3) → ranks (1,3,2) = 132. ✓
  • Positions (2,4,7): values (2,4,7) → ranks (1,2,3) = 123. ✗
  • …(checking all 35 triples)…

Total occurrences of 132: 4. So p(132, 6214537) = 4/35. ✓

Key property The 6 pattern densities for patterns of length 3 always sum to 1 — they form a probability distribution on S₃. This is why we can compute total variation distance between two permutations' density profiles.

Deducing Shorter Patterns from Longer Ones

Marginal Consistency — Recovering Length-2 from Length-3 Densities If you know all six length-3 pattern densities, you can recover the length-2 densities. For the pattern 12 (increasing pair):
p(12, σ) = [3·p(123) + 2·p(132) + 2·p(213) + 1·p(231) + 1·p(312) + 0·p(321)] / 3
And for 21 (decreasing pair): p(21, σ) = 1 − p(12, σ).
Why this works

Pick a random 2-element subset {i,j} (i<j). We want P(σ(i)<σ(j)) = p(12,σ). Now extend this to a random 3-element subset by adding a third uniformly random position. The triple has a pattern in S₃. How many increasing pairs does each S₃ pattern contain?

Pattern123132213231312321
# increasing pairs322110

A random triple contains 3 random pairs. The expected number of increasing pairs = 3·p(12,σ). But it also equals the weighted sum above. Dividing by 3 gives the formula. ✓

This generalises: all shorter-pattern densities are determined by longer-pattern densities. In the limit, the full permuton is determined by its pattern densities — if you know all pattern densities consistently, you know the permuton.

Example — 1324 (n=4)

All length-3 densities for 1324 = (1,3,2,4):

Density1/21/41/4000

Note: 1324 avoids 231 and 321, so those densities are 0.

Recovering p(12) from these: [3·(1/2) + 2·(1/4) + 2·(1/4) + 0 + 0 + 0] / 3 = [3/2 + 1/2 + 1/2] / 3 = (5/2)/3 = 5/6. ✓

03Permuton — Formal Definition

Definition — Permuton A permuton is a probability measure μ on [0,1]² with uniform marginals:
μ([0,1] × [0,y]) = y and μ([0,x] × [0,1]) = x for all x,y ∈ [0,1]
Both the x-projection and the y-projection of μ are the uniform distribution on [0,1].

Reading the diagram
  • Step 1 (scatter plot): element at position i with value σ(i) becomes a dot at (i, σ(i)) in ℤ² space.
  • Step 2 (matrix): each dot becomes a 1 in an n×n binary matrix. Each row and column has exactly one 1 (it's a permutation matrix).
  • Step 3 (permuton): rescale to [0,1]². Position i maps to the interval [(i−1)/n, i/n] on the x-axis. Value σ(i) maps to [(σ(i)−1)/n, σ(i)/n] on the y-axis. Each of the n boxes gets mass 1/n. This is the step permuton μ_σ.
  • As n→∞: if permutations have a "global structure" that stabilizes, the step permutons converge to a continuous measure — the permuton limit.

CDF of a Permuton

CDF of a Permuton The cumulative distribution function of a permuton μ is:
G(x, y) := μ([0,x] × [0,y])
the probability that a random point drawn from μ falls in the lower-left rectangle [0,x]×[0,y].

The uniform marginal conditions become:
G(x, 1) = x (x-marginal uniform) G(1, y) = y (y-marginal uniform)
If μ has a density g(x,y) (i.e., μ is absolutely continuous), then:
∫₀¹ g(x,y) dy = 1 for a.e. x and ∫₀¹ g(x,y) dx = 1 for a.e. y
Going from Permuton to Permutation Distribution Every permuton μ induces a probability distribution on Sₙ for each n:
  1. Sample n points (x₁,y₁),…,(xₙ,yₙ) i.i.d. from μ.
  2. Sort by x-coordinate: relabel so x₁ < x₂ < ··· < xₙ.
  3. Record the permutation given by the y-coordinates: σ(i) = rank of yᵢ among {y₁,…,yₙ}.
This gives a random permutation in Sₙ. As n→∞, the step permuton of this random permutation converges back to μ.
Example — permuton for pattern 231

The permuton giving the "average shape" of a 231-pattern is concentrated near the three points (0,½), (½,1), (1,0) in [0,1]² — reflecting the relative order: middle value comes first, then large, then small.

04Convergence of Permutation Sequences

Definition — Convergence of Permutations A sequence of permutations σ₁, σ₂, σ₃, … (with |σₙ| → ∞) converges to a permuton μ if their step permutons μ_{σₙ} converge weakly to μ — equivalently, if the CDF converges pointwise:
G_{σₙ}(x, y) → G_μ(x, y) for all (x,y) ∈ [0,1]²
Equivalently: for every fixed pattern π ∈ Sₖ, the pattern densities converge:
p(π, σₙ) → p(π, μ)
What "global structure" means

As n grows, a permutation of length n has more and more local complexity. But if the large-scale shape stabilises — meaning the fraction of triples, quadruples, etc. forming each pattern converges — then the sequence converges. The permuton captures exactly this global shape.

Think of it like a blurry photograph: the fine details (which specific positions have which values) wash out as n→∞, but the overall pattern (how values are distributed spatially) remains.

Two Key Convergence Examples

Example 1 — Identity sequence converges to the diagonal

Sequence: 1, 12, 123, 1234, 12345, …

For every n, σₙ = (1,2,…,n) is the identity permutation. Its step permuton places mass 1/n at each box along the diagonal: [(i−1)/n, i/n] × [(i−1)/n, i/n].

As n→∞, these boxes shrink and fill in the diagonal line y=x uniformly. The limit is the diagonal permuton — the measure supported on {(x,x) : x ∈ [0,1]}.

Pattern densities: p(123) → 1, all others → 0. ✓ (The identity is always increasing.)

Example 2 — Random sequence converges to the uniform permuton

Sequence: 132, 4123, 35124, …

Two extreme permutons
  • Diagonal (identity): supported on y=x. Pattern densities: p(12…k) = 1, all others = 0 for every k.
  • Anti-diagonal (reverse): supported on y=1−x. Pattern densities: p(k…21) = 1, all others = 0.
  • Uniform (gray square): p(π) = 1/k! for all π ∈ Sₖ, for all k.

05The Random Walk Permuton

Setup Take n−1 steps of a random walk: X₁=0, Xᵢ₊₁ = Xᵢ + Δᵢ, where Δ₁,…,Δₙ₋₁ are i.i.d. with a symmetric, continuous distribution (e.g. standard normal). The permutation σ is induced by ranking the values X₁,…,Xₙ.
Pattern Densities for the Random Walk Permuton For any triple of positions (consecutive or not), the six S₃-pattern probabilities are:
p(123) = p(321) = 1/4 p(132) = p(231) = p(213) = p(312) = 1/8
The monotone patterns (123 and 321) each get probability 1/4. The four non-monotone patterns each get 1/8.
Proof — for a 3-step walk

Let X₁=0, X₂=Δ₁, X₃=Δ₁+Δ₂ with Δ₁,Δ₂ i.i.d. symmetric. The pattern of (X₁,X₂,X₃) is determined by which values are in increasing or decreasing order. There are 4 cases based on signs of Δ₁ and Δ₂:

(Δ₁>0, Δ₂>0): X₁<X₂<X₃ → pattern 123. P = 1/4.
(Δ₁<0, Δ₂<0): X₁>X₂>X₃ → pattern 321. P = 1/4.
(Δ₁>0, Δ₂<0): X₂ is larger than both X₁ and X₃. P(Δ₁>0,Δ₂<0)=1/4.
Within this case, by symmetry of |Δ₁| and |Δ₂|: P(|Δ₁|>|Δ₂|)=P(|Δ₁|<|Δ₂|)=1/2.
If |Δ₁|>|Δ₂|: X₁<X₃<X₂ → pattern 132. P = 1/4 × 1/2 = 1/8.
If |Δ₁|<|Δ₂|: X₃<X₁<X₂ → pattern 231. P = 1/4 × 1/2 = 1/8.
(Δ₁<0, Δ₂>0): By symmetry: patterns 213 and 312 each with P = 1/8.

Sum: 1/4 + 1/4 + 4×(1/8) = 1. ✓

This structure holds for any triple of positions in a long random walk, not just consecutive ones. The random walk permuton has this as its length-3 pattern density profile.

Lecture example The lecture noted: P(X₂ > X₁) = 1/2 (obvious), and naively one might expect P(123) = 1/2 × 1/2 = 1/4 since each step is symmetric. But this is NOT the right reasoning — the events "X₁<X₂" and "X₂<X₃" are NOT independent (they share X₂). The correct answer P(123)=1/4 happens to equal 1/4 but for the right reason: P(Δ₁>0 AND Δ₂>0) = 1/4 by independence of Δ₁,Δ₂. For P(231)=1/8, the reasoning goes through the sign decomposition above.

06Support of a Permuton

Definition — Support The support of a permuton μ is the smallest closed subset of [0,1]² with μ-probability 1:
supp(μ) = ⋂ {C ⊆ [0,1]² : C closed and μ(C) = 1}
Equivalently, a point (x,y) is in supp(μ) iff every open neighborhood of (x,y) has positive μ-measure.
Examples
PermutonSupportDescription
Identity (diagonal){(x,x) : x ∈ [0,1]}The main diagonal — a 1D curve in [0,1]²
Reverse (anti-diagonal){(x,1−x) : x ∈ [0,1]}The anti-diagonal from (0,1) to (1,0)
Uniform (gray square)[0,1]²The entire square — full support
Step permuton of σ ∈ Sₙn boxes of size 1/n × 1/nFinite support — n rectangular boxes along the matrix positions
Random walk permutonA specific region of [0,1]²Related to the arc-sine distribution
Support tells you what's possible The support describes which (position, value) combinations occur with positive probability in the limit. A permuton supported on the diagonal means that in the limit, large elements tend to be at the "correct" positions — the permutation looks like the identity. A permuton with full support means every region of the unit square is visited with positive probability.

07Basis of a Permutation Class (HW May 20)

Recall — Permutation Class A permutation class C is a set of permutations closed downward under pattern containment: if σ ∈ C and π is a pattern of σ, then π ∈ C. Every permutation class has the form Av(F) for some set of forbidden patterns F.
Theorem — Unique Basis (HW May 20) For every permutation class C, there exists a unique antichain A (called the basis of C) such that C = Av(A).

The basis is: A = {σ ∉ C : every proper pattern of σ is in C} — the minimal permutations not in C.
Proof

Existence — C = Av(A).

C ⊆ Av(A): Suppose σ ∈ C contains some π ∈ A. Then π is a pattern of σ. Since C is downward closed, π ∈ C. But π ∈ A means π ∉ C — contradiction. So σ ∈ C avoids every element of A, i.e. σ ∈ Av(A). ✓
Av(A) ⊆ C: Suppose σ ∉ C. By well-foundedness of the pattern order (no infinite descending chains), there exists a minimal subpermutation of σ that is not in C. Call it π. Every proper pattern of π lies in C (by minimality of π). So π ∈ A. And π is a pattern of σ, so σ contains A → σ ∉ Av(A). ✓

A is an antichain. Suppose π, τ ∈ A with π a proper pattern of τ. Since τ ∈ A, every proper pattern of τ is in C — so π ∈ C. But π ∈ A means π ∉ C. Contradiction. So no element of A is a pattern of another. ✓

A is unique. Suppose B is another antichain with C = Av(B). Let π ∈ A. Since π ∉ C = Av(B), π contains some β ∈ B. But every proper pattern of π is in C = Av(B), so it avoids B, meaning β is not a proper pattern of π — so β = π. Thus π ∈ B. Similarly B ⊆ A. So A = B. ✓

Examples of Bases
  • Av(231): basis = {231}. (Stack-sortable permutations.)
  • Av(321): basis = {321}. (2-increasing permutations.)
  • All permutations: basis = ∅ (no forbidden patterns).
  • Only the identity permutation (1,2,…,n) for each n: basis = {21}. (Avoiding any descent.)
  • Av(231, 321): basis = {231, 321} — an antichain since neither is a pattern of the other.

08Four Modes of Convergence (HW May 22)

The HW asks for a random sequence of points X₁, X₂, X₃, … on [0,1] illustrating four different convergence behaviors. There are two types of convergence to distinguish:

Two types of convergence
  • Almost sure (a.s.) convergence: P(Xₙ → X) = 1. Every sample path converges. The stronger notion.
  • Convergence in distribution: P(Xₙ ≤ t) → P(X ≤ t) for all t. The CDFs converge. The weaker notion.
Key relationship: a.s. convergence ⟹ convergence in distribution, but NOT vice versa.
(1)
Converges a.s. to a fixed point

Every sample path converges, and they all converge to the same limit.

Example: Xₙ = Uⁿ where U ~ Uniform[0,1].

Since P(U < 1) = 1, we have Uⁿ → 0 almost surely. Every sample path converges to 0.

Xₙ = Uⁿ, U ~ Unif[0,1] → 0 a.s.
(2)
Converges a.s., but not to the same point

Every sample path converges, but different paths converge to different limits.

Example: Xₙ = U for all n, where U ~ Uniform[0,1] is fixed at the start.

Each sample path is constant (Xₙ = U always) → converges. But the limit U is random — different runs give different limits.

Xₙ = U for all n, U ~ Unif[0,1] (random constant)
(3)
Distributions converge, but paths don't

The CDF of Xₙ stabilizes, but individual sample paths oscillate without converging.

Example: Xₙ = Uₙ where U₁, U₂, … are i.i.d. Uniform[0,1].

Each Xₙ ~ Uniform[0,1] — distribution is always the same, so it trivially converges in distribution. But each Uₙ is independent of the others, so sample paths jump around [0,1] forever without converging.

Xₙ = Uₙ i.i.d. Unif[0,1]. Distrib. ✓, paths ✗
(4)
Neither paths nor distributions converge

Sample paths don't converge and the distribution itself oscillates without a limit.

Example: Xₙ = n mod 2 (deterministic: 0, 1, 0, 1, …).

Paths: each path is 0,1,0,1,… — no limit. Distributions: alternate between δ₀ (all mass at 0) and δ₁ (all mass at 1) — no weak limit since δ₀ ≠ δ₁.

Xₙ = n mod 2 → 0,1,0,1,… No limit anywhere.
Connection to permutons This is not just abstract probability theory — it's directly about how permuton sequences behave. A sequence of permutations can converge in distribution (as permutons) even when no individual permutation "converges" in any simple sense. Cases (3) and (4) show precisely when distributional convergence can fail.

σ = (1,4,2,3,5), n=5. Total triples: C(5,3) = 10.

p(123) = 5/10 = 1/2   p(132) = 2/10 = 1/5   p(213) = 2/10 = 1/5
p(231) = 0   p(312) = 1/10   p(321) = 0

Q2For the uniform permuton, what are the length-3 pattern densities?▼
Solution
Uniform permuton = Lebesgue measure on [0,1]². Sample n points uniformly, sort by x, record y-ranks → uniform random permutation.
For a uniform random permutation: all 3! = 6 orderings of any 3 elements are equally likely. So each pattern has density 1/6.

p(π) = 1/6 for all π ∈ S₃. The uniform permuton is maximally "mixed".

Q3Describe the step permuton for σ = (3,1,4,2) in [0,1]². Where are the four boxes?▼
Solution
n=4, each box has size 1/4 × 1/4 and mass 1/4.
Box for position i, value σ(i): x ∈ [(i−1)/4, i/4], y ∈ [(σ(i)−1)/4, σ(i)/4].
i=1, σ=3: x ∈ [0, 0.25], y ∈ [0.5, 0.75]

Q4Show that knowing all length-3 pattern densities determines the length-2 density p(21). Give the formula.▼
Solution
Formula for p(12): each S₃ pattern contributes to p(12) based on how many increasing pairs it has: 123→3, 132→2, 213→2, 231→1, 312→1, 321→0. Weighted average divided by 3 gives p(12).
p(12) = [3p(123) + 2p(132) + 2p(213) + p(231) + p(312)] / 3
Then: p(21) = 1 − p(12).

p(21) = 1 − [3p(123) + 2p(132) + 2p(213) + p(231) + p(312)] / 3

Q5Give an example of a permutation class whose basis contains infinitely many elements.▼
Solution
Example: Let C = {σ : σ is a permutation of some [n] that has at most one descent}. This is the class of "layered" permutations of the form (k, k−1, …, 1, k+1, k+2, …, n) for various k — a single increasing run after a decreasing prefix.
Actually, a cleaner example: the class of stack-sortable permutations sortable in 2 passes. Its basis is infinite — there's no finite set of forbidden patterns that characterizes it.
Or even simpler: the class of permutations whose longest increasing subsequence has length ≤ 2. These avoid 321 and... wait, that's just Av(321), which has basis {321} — finite.
Instead: the class of permutations avoiding 12...k for all k simultaneously. This class = {only decreasing permutations}, but actually this has basis {12} (one element).
Real example: the class of separable permutations has an infinite basis.

The basis can be infinite. E.g., the class of permutations not containing a monotone subsequence of length k for ANY k has infinitely many minimal forbidden patterns.

Q6For the random walk permuton, what is p(12)? Use the length-3 marginal formula.▼
Solution
Length-3 densities for random walk: p(123)=p(321)=1/4, all others=1/8.
Apply marginal formula: p(12) = [3·(1/4) + 2·(1/8) + 2·(1/8) + 1·(1/8) + 1·(1/8) + 0·(1/4)] / 3 = [3/4 + 1/4 + 1/4 + 1/8 + 1/8] / 3 = [3/4 + 1/2 + 1/4] / 3 = [3/2] / 3 = 1/2

Makes sense! The random walk is symmetric, so P(step up) = P(step down) = 1/2. The probability that any two values are in increasing order is 1/2.

p(12) = 1/2 for the random walk permuton. ✓

Q7Construct a random sequence on [0,1] that converges in distribution to a non-trivial limit, but fails to converge almost surely.▼
Solution
Use the oscillating example: Let U ~ Uniform[0,1] (fixed at start). Define Xₙ = U if n is even, 1−U if n is odd.
Distribution: Both U and 1−U are Uniform[0,1]. So Xₙ ~ Uniform[0,1] for all n. Distributions converge (trivially, always the same). ✓
Almost sure behavior: For any fixed U ≠ 1/2, the sequence alternates between U and 1−U ≠ U forever. Sample paths do not converge (except on the probability-0 event U=1/2). ✗

Xₙ = U if n even, 1−U if n odd. Distribution ≡ Unif[0,1]. Paths don't converge a.s.

Q8What is the basis of the class Av(132, 231)?▼
Solution
Is {132, 231} already an antichain? Neither 132 is a pattern of 231 nor 231 is a pattern of 132 (they're both length 3 and neither equals the other). Yes, antichain. ✓
Is every permutation not in Av(132,231) containing 132 or 231? Yes by definition — Av(132,231) = permutations avoiding both. The minimal forbidden permutations ARE 132 and 231 since both are length 3 and all length-1 and length-2 permutations are in Av(132,231).

Basis = {132, 231}. Both are minimal — all patterns of length <3 are trivially avoided.

HW — Due Wednesday May 20

Problem: Prove that every permutation class C has a unique antichain basis A with C = Av(A).

Key steps:

  1. Define A = {minimal permutations not in C}.
  2. C ⊆ Av(A): if σ ∈ C contains π ∈ A, then downward closure gives π ∈ C, contradicting π ∈ A.
  3. Av(A) ⊆ C: if σ ∉ C, take minimal sub-permutation π ∉ C — then π ∈ A and π ⊆ σ, so σ ∉ Av(A).
  4. A is antichain: if π were a proper pattern of τ (both in A), minimality of τ gives π ∈ C, contradicting π ∈ A.
  5. Uniqueness: if B is another antichain with C = Av(B), show A ⊆ B and B ⊆ A by chasing minimality.

Problem: Describe random sequences X₁,X₂,… on [0,1] illustrating each behavior:

  2. Converges a.s., not always same point: Xₙ = U for all n. Each path is constant (converges to U), but U is random over [0,1].
  3. Distributions converge, paths don't: Xₙ = Uₙ i.i.d. Unif[0,1]. Distribution is always Unif[0,1] (converges trivially). Paths oscillate — no a.s. limit.
  4. Neither converges: Xₙ = n mod 2. Deterministic sequence 0,1,0,1,… — no limit. Distributions alternate δ₀ and δ₁ — no weak limit.

  • Prob. measure on [0,1]² with uniform marginals
  • G(x,y) = μ([0,x]×[0,y]); G(x,1)=x, G(1,y)=y
  • σ → step permuton: mass 1/n at box [(i−1)/n,i/n]×[(σ(i)−1)/n,σ(i)/n]
  • Diagonal permuton: supp = {y=x}; p(123…k)=1
  • Uniform permuton: full [0,1]²; p(π)=1/k! ∀π∈Sₖ
Random Walk Permuton
  • n−1 symmetric i.i.d. steps
  • p(123)=p(321)=1/4; p(132)=p(231)=p(213)=p(312)=1/8
  • p(12) = 1/2 (symmetric steps)
Convergence
  • Sequence converges to μ iff CDFs G_{σₙ}(x,y) → G_μ(x,y)
  • Equivalently: all pattern densities p(π,σₙ) → p(π,μ)
  • 1,12,123,… → diagonal permuton
  • Uniform random perms → uniform permuton
Support
  • supp(μ) = smallest closed set with μ-prob 1
  • = ⋂{closed C : μ(C)=1}
Basis of Permutation Class
  • A = minimal permutations NOT in C
  • C = Av(A), A is an antichain, A is unique
  • Proof: minimality gives existence and antichain; uniqueness by inclusion
Four Convergence Modes
  • (1) a.s. fixed: Xₙ=Uⁿ → 0
  • (2) a.s. random: Xₙ=U (constant U~Unif)
  • (3) distrib. only: Xₙ=Uₙ i.i.d. Unif
  • (4) neither: Xₙ = n mod 2
  • a.s. ⟹ in distribution (not vice versa)
